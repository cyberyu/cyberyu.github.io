doc_to_label(conn, out, **kwargs)

source code 

Apply active learning classifier to predict relevance labels.

Parameters:
  • conn (psycopg2 connection object) - The established connection object to the Google Search Result database
  • out (Python dictionary object) - A python dictionary {"docs": docs, "labels": label, "srids":srids} containing all the training and test data. Docs are a list of text. Labels are a list of relevance flags used for training. srids is a list of document ids.
  • kwargs - remaining keyword arguments are passed to readDB functon, see below

    Args:

  • textfield (list object) - the list of Google Search Result table names to be concatenated as text field. Example: ['fulltext', 'srtitle']
  • training_srids (list object) - the list of document ids used as training data
  • test_srids (list object) - the list of document ids used as test data
Returns:
Python data discionary object {"scores": ToConfirmScore, "srids":ToConfirm_Id}. ToConfirmScore is the list of relevance scores that require user to confirm. Srids is the list of document ids that require user confimation. The predicted flag and relevance scores for all the test data will will be automatically updated in the table.
